# Shoyeb AI Portfolio Assistant

A FastAPI-based web application that serves as an AI-powered portfolio assistant for Shoyeb ShaikhChand Chaudhari. Users can interact with a chat interface to ask questions about his skills, projects, experience, and education, with responses generated by Google's Gemini AI model.

## Description

This project is a portfolio assistant chatbot built with FastAPI, featuring a modern web UI. It leverages LangChain's Google Generative AI integration to provide intelligent responses about Shoyeb's professional background. All Q&A interactions are stored in a SQLite database for memory persistence.

## Features

- **Interactive Chat Interface**: Web-based chat UI for asking questions about Shoyeb's portfolio
- **AI-Powered Responses**: Uses Google's Gemini 2.5 Flash model for generating answers
- **Memory Persistence**: Stores all questions and answers in a SQLite database
- **Responsive Design**: Clean, modern UI with sidebar profile and chat area
- **CORS Enabled**: Supports cross-origin requests for API access
- **Static File Serving**: Serves CSS and images from the static directory
- **Template Rendering**: Uses Jinja2 templates for HTML responses

## Technologies Used

- **Backend**: FastAPI, Python
- **AI Integration**: LangChain Google Generative AI (Gemini)
- **Database**: SQLAlchemy with SQLite
- **Frontend**: HTML, CSS, JavaScript
- **Templating**: Jinja2
- **Environment Management**: python-dotenv
- **Other**: Pydantic for data validation, Uvicorn for ASGI server

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd <project-directory>
   ```

2. Create a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Set up environment variables:
   - Create a `.env` file in the root directory
   - Add your Google Gemini API key:
     ```
     GEMINI_API_KEY=your_api_key_here
     ```

## Usage

1. Start the FastAPI server:
   ```bash
   uvicorn app:app --reload
   ```

2. Open your browser and navigate to `http://localhost:8000`

3. Interact with the chat interface:
   - Ask questions about Shoyeb's skills, projects, experience, or education
   - The AI will respond in third person with formatted, humorous answers
   - All conversations are saved to the database

## API Endpoints

### GET /
- **Description**: Serves the main web interface
- **Response**: HTML page with chat UI

### POST /ask
- **Description**: Submits a question to the AI assistant
- **Request Body**:
  ```json
  {
    "question": "What are Shoyeb's main skills?"
  }
  ```
- **Response**:
  ```json
  {
    "answer": "Shoyeb's main skills include Java, Python, SQL, and more..."
  }
  ```

## Database

The application uses SQLite (`memory.db`) with a single table `QAMemory`:
- `id`: Primary key (Integer)
- `question`: User's question (Text)
- `answer`: AI-generated answer (Text)
- `created_at`: Timestamp (DateTime)

The database is automatically initialized on application startup.
